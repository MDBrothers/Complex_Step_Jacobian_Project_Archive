#Title: Experimental Input, Ouput and Reduced Data

#Description: 

Contained within experimental_input_and_output.tar.bz2 the mesh files and
input xml files necessary to reproduce the results shown in the research
paper. Refer to Peridigm user documentation for use instructions. 

example.jou is an example input file which demonstrates how the input meshes
were created for the study using the 'Cubit' software.
